# Setup for Coder Development Environment

## Quick Start

1. Install and configure zsh
    ```console
    $ sudo apt install zsh
    $ sudo chsh -s /bin/zsh $(whoami)
    ```
1. Install `aws`
    ```console
    $ curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
    unzip awscliv2.zip
    sudo ./aws/install
    ```
